# Architecture Decision Records (ADR)

Los ADR documentan decisiones técnicas importantes del proyecto, su contexto y consecuencias. Siguen el formato propuesto por [Michael Nygard](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions.html).

## Estructura

Cada ADR sigue esta plantilla:

- **Título**: Número secuencial y nombre descriptivo
- **Contexto**: Por qué se necesita esta decisión
- **Decisión**: Qué se decidió
- **Consecuencias**: Impacto positivo y negativo
